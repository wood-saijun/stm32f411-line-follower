# stm32f411-line-follower

## 项目简介
本项目为电子信息专业大三学生的实践项目，旨在实现一个基于 STM32F411 的巡线小车。

## 项目基础
以数模电、微机原理为理论指导，参考江科大、keysking 的例程搭建项目基础知识。

## 项目实现工具
- 硬件设计：嘉立创 EDA
- 软件开发：STM32CubeMX + CLion + Gemini

## 器材清单
- HC-06 蓝牙模块 × 1
- MPU6050 陀螺仪 × 1
- STM32F411CEU6 主控 × 1
- OLED × 1
- TB6612 电机驱动 × 1
- 灰度传感器 × 1
- 小车底座 × 1
- 带编码器电机 × 2
- 蜂鸣器、电阻、电容、按键若干
