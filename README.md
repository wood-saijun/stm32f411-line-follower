# stm32f411-line-follower
本项目为电子信息专业大三学生的实践项目：
以数模电，微机原理为理论指导，以江科大，keysking例程搭建项目基础知识
项目实现工具：硬件pcb：嘉立创eda绘制，软件：stm32cubemx+clion+gemini
器材清单：HC-06蓝牙模块一个，mpu6050陀螺仪一个，stm32f411cet6主控一个，oled一个，tb6612电机驱动一个，基于数字通信的灰度传感器一个，小车底座一个（带霍尔编码器的电机）电容，电阻，按键若干。
学习网站：波特律动（代码，文档，视频教程），江协科技（代码，ppt，视频教程），expert电子实验室（ppt，视频教程)
